# Projecto Ingeniería de Software 2 - 2024 UdeC

## Descripción
Proyecto frontend en React para la asignatura de Ingeniería de Software 2 de la Universidad de Concepción.

## Tecnologías
- vite
- react
- react-router-dom

## Instalación
1. Clonar el repositorio
2. Instalar las dependencias
```bash
npm install
```

# Uso
```bash
npm start
```
